# Sanity CMS সেটআপ গাইড - বাংলা

## ১. CMS কী এবং কেন?

**CMS = Content Management System** - ব্লগ পোস্ট তৈরি করার জন্য।
- আগে: স্ট্যাটিক ডেটা ফাইলে রাখা ছিল
- এখন: **Sanity CMS-এ রাখা** (ক্লাউডে)
- সুবিধা: যেকোনো জায়গা থেকে এডিট করতে পারবেন

---

## ২. Studio অ্যাক্সেস করা (লোকাল)

### লোকাল Studio চালু করা
```bash
npm --prefix studio run dev
```
এরপর খুলুন: **http://localhost:3333**

### লগইন করুন
- Google / GitHub / Email দিয়ে লগইন করুন
- প্রথম লগইন একটু সময় লাগতে পারে

---

## ৩. ব্লগ পোস্ট তৈরি করা

### ধাপে ধাপে:

**Step 1:** Studio ড্যাশবোর্ডে "Create" বাটন ক্লিক করুন (বাম দিকে)

**Step 2:** "Blog Post" সিলেক্ট করুন

**Step 3:** ফর্ম পূরণ করুন:

| ফিল্ড | বর্ণনা | উদাহরণ |
|------|-------|--------|
| **Title** | পোস্টের শিরোনাম | "Next.js দিয়ে আধুনিক ওয়েবসাইট তৈরি" |
| **Slug** | URL-friendly শনাক্ত | "nextjs-modern-website" (স্বয়ংক্রিয়) |
| **Excerpt** | সংক্ষিপ্ত বর্ণনা (20-240 অক্ষর) | "Next.js এবং React ব্যবহার করে..." |
| **Published At** | প্রকাশের তারিখ/সময় | আজকের তারিখ |
| **Read Minutes** | পড়তে কত মিনিট লাগবে | 5-10 |
| **Category** | ক্যাটাগরি বেছে নিন | Engineering / Design / Field Notes |
| **Tags** | ট্যাগ যোগ করুন | react, nextjs, web |
| **Accent Color** | পোস্টের থিম রঙ | কোন রঙ ইচ্ছে করুন |
| **Author** | লেখকের নাম | আপনার নাম |
| **Body** | পোস্টের মূল কন্টেন্ট | সম্পূর্ণ পোস্ট টেক্সট |

**Step 4:** "Publish" বাটন ক্লিক করুন

✅ **হয়ে গেছে!** আপনার পোস্ট এখন লাইভ।

---

## ৪. ফ্রন্টএন্ডে দেখা যাচ্ছে কিনা?

পোস্ট পাবলিশ করার ২-৫ সেকেন্ডের মধ্যে:
- **http://localhost:5000/blog** - সব পোস্ট দেখবেন
- **http://localhost:5000/blog/nextjs-modern-website** - সেই পোস্টের বিস্তারিত

---

## ৫. সেই API URL কী?

```
https://<sanity-project-id>.api.sanity.io/v<api-version>/data/query/<dataset>?query=...
```

**এটি কী করে:**
- আপনার ব্লগ পোস্ট ফেচ করার URL
- Frontend এটি ব্যবহার করে ডেটা পায়
- যেকোনো ডোমেইন থেকে কাজ করে (লোকাল, লাইভ, সবকিছু)

**প্রজেক্ট ID:** `<sanity-project-id>`
**ডেটাসেট:** `<dataset>`

## ৬. লাইভ ডোমেইনে কনফিগারেশন

### বিকল্প A: শুধু ফ্রন্টএন্ড লাইভ (সহজ - বেশিরভাগ সময়)
```
Frontend: yourdomain.com ✅ কাজ করবে
Studio (CMS): localhost:3333 (শুধু আপনার কম্পিউটার)
```
- ব্যাকএন্ড কনফিগার করা লাগবে না
- আপনি লোকালি পোস্ট তৈরি করবেন, দেশব্যাপী মানুষ দেখবে

### বিকল্প B: লাইভ ডোমেইনেও Studio (বিকল্প)
```
Frontend: yourdomain.com ✅
Studio: cms.yourdomain.com ✅
```
- এক্সট্রা সেটআপ প্রয়োজন
- যেকোনো জায়গা থেকে Studio অ্যাক্সেস করতে পারবেন

## ৭. ফ্রন্টএন্ড লাইভ করার আগে যাচাই করুন

```bash
# ১. ফ্রন্টএন্ড বিল্ড
npm --prefix web run build

# ২. ফ্রন্টএন্ড টেস্ট চালান
npm --prefix web run test

# ৩. স্ট্যাটিক আউটপুট দেখুন
ls -la web/.next/static
```

সবকিছু ✅ সফল হলে লাইভ ডেপ্লয় করতে পারেন।

## ৮. ট্রাবলশুটিং

### সমস্যা: Studio খুলছে না
```bash
npm --prefix studio run dev
# এবং localhost:3333 খুলুন
```

### সমস্যা: ব্লগ পোস্ট ফর্ম খালি দেখাচ্ছে
```bash
# Studio বন্ধ করুন এবং পুনরায় চালু করুন
npm --prefix studio run dev
```

### সমস্যা: ফ্রন্টএন্ডে পোস্ট দেখাচ্ছে না
```bash
# Sanity API টেস্ট করুন
curl "https://<sanity-project-id>.api.sanity.io/v<api-version>/data/query/<dataset>?query=%7B%22query%22:%22%2A%5B_type%20%3D%3D%20%27blogPost%27%5D%7B%20title,%20slug,%20excerpt%20%7D%22%7D"
```

## ৯. আপনার করণীয়

✅ **এখনই করুন:**
1. Studio খুলুন: `http://localhost:3333`
2. লগইন করুন
3. Create → Blog Post ক্লিক করুন
4. কমপক্ষে ২-৩টি পোস্ট তৈরি করুন
5. `http://localhost:5000/blog` খুলে দেখুন

📝 **পরে (যখন লাইভ করতে চাইবেন):**
1. আপনার লাইভ ডোমেইন বলুন
2. আমি লাইভ ডেপ্লয় সেটআপ করব
3. Frontend লাইভ চলবে
