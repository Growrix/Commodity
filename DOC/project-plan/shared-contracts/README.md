# Shared Contracts

This folder is the system blueprint for the Commodity Trading Company Website. It locks the cross-role decisions that frontend, backend, CMS, analytics, security, and QA must all share.

## Owns

- Surface boundaries and route contract
- Public content ownership model
- Operational entity inventory and lifecycle states
- Global invariants and explicit non-goals
- Cross-role implementation sequence
