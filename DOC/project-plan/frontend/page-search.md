# Search Page

## Metadata

- Route: `/search`
- Template type: utility discovery page
- Primary audiences: all public visitors
- Primary CTA: Open Matching Result
- Secondary CTA: Contact Trade Desk

## Goal

Provide a fast, readable discovery tool for products, insights, markets, and trust documents without creating SEO-noisy thin pages.

## Section Plan

### 1. Search Header
- Purpose: Make search the primary action immediately.
- Content: input, helper text, optional popular queries.

### 2. Filter And Scope Bar
- Purpose: Narrow results by content type.
- Content: products, insights, markets, trust documents, capabilities.

### 3. Results Groups
- Purpose: Present grouped, scannable results.
- Content: cards or list rows grouped by type.

### 4. Zero-State Guidance
- Purpose: Avoid dead ends.
- Content: suggested sectors, product families, insights, and contact CTA.

### 5. Support CTA
- Purpose: Preserve commercial conversion.
- Content: contact trade desk or browse top categories.

## State Coverage

- Default: query entry and grouped results.
- Loading: result-row skeletons.
- Empty: zero-state guidance with suggested destinations.
- Success: not applicable beyond result click-through.
- Error: fallback message with browse links and contact CTA.
