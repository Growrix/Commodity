# 404 Page

## Metadata

- Route: unmatched public route
- Template type: utility recovery page
- Primary audiences: all visitors
- Primary CTA: Return Home
- Secondary CTA: Search The Site

## Goal

Recover lost visitors quickly and steer them toward high-value destinations rather than ending the session.

## Section Plan

### 1. Error Header
- Purpose: Explain the issue clearly without blame.
- Content: short headline, plain-language explanation.

### 2. Primary Recovery Actions
- Purpose: Provide immediate next steps.
- Content: home CTA, search CTA, products CTA.

### 3. Suggested Destinations
- Purpose: Retain visitors through high-value entry points.
- Content: products, markets, insights, contact, trust center.

### 4. Contact Fallback
- Purpose: Support users who need human help.
- Content: general contact CTA.

## State Coverage

- Default: recovery-focused utility page.
- Loading: not applicable.
- Empty: not applicable.
- Success: not applicable.
- Error: not applicable beyond standard route handling.
