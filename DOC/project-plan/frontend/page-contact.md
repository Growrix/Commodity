# Contact Page

## Metadata

- Route: `/contact`
- Template type: conversion and support page
- Primary audiences: all commercial visitors
- Primary CTA: Send Inquiry
- Secondary CTA: Use Specialized Forms

## Goal

Provide a low-friction contact path while still steering users into more structured conversion forms when appropriate.

## Section Plan

### 1. Hero
- Purpose: Set expectations for contact and response.
- Content: contact summary, response SLA, CTA.

### 2. Contact Path Selector
- Purpose: Route users to quote, sourcing, supplier, or general inquiry.
- Content: action cards with one-line guidance.

### 3. General Inquiry Form
- Purpose: Capture non-specialized contact requests.
- Content: name, company, topic, message, region.

### 4. Regional Offices
- Purpose: Reinforce global presence.
- Content: office cards, time zones, regional points of contact.

### 5. Assurance Note
- Purpose: Reduce hesitation.
- Content: confidentiality, next steps, document readiness note.

## State Coverage

- Default: contact page with routing cards and general form.
- Loading: form and office-card skeletons.
- Empty: if no offices are listed, preserve form and global contact channel.
- Success: contact request acknowledgement.
- Error: preserve form state and route users to alternate contact method.
